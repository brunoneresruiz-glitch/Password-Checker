# PASSWORD STRENGTH CHECKER

A simple cybersecurity tool that analyses password strength using real world security criteria and entropy calculation, built with **Python**.

## What it actually does
- Detects commum and weak passwords.
- Enforces minumum length
- Checks for:
  - Uppercase and lowercase letters
  - Numbers and special characters
- Calculates password entropy ( in bits)
- Score from 0 to 100 with personalised feedback

## The reason why this project:

This project was built while playing with **Python** to show my understanding of:
- Regular impressions (regex)
- Basic **Cybersecurity** principles
- Password security evaluation logic

## How to run
bash python main.py
